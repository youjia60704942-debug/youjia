📱 如何取得員工的 LINE User ID

步驟：

  1. 讓員工掃描 LINE Bot QR Code 加好友
  2. 員工傳送任意訊息給 Bot
  3. 到 LINE Developers 控制台 → Messaging API → Webhook 查看收到的 userId
  4. 或設定 Webhook 自動回覆 User ID 給員工
💡 Bot QR Code 在 LINE Developers → 您的 Channel → Messaging API 分頁，分享給員工掃描加好友。User ID 格式為 U 開頭的 32 碼英數字。
